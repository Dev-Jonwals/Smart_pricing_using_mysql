The project simulates a real-world pricing analytics workflow used by quick commerce companies like Blinkit, Zepto, and Swiggy Instamart.

---

##  Business Problem
Quick commerce platforms frequently apply blanket discounts, which can:
- Reduce margins unnecessarily
- Over-discount loyal and low price-sensitive users
- Miss growth opportunities for price-sensitive cohorts

This project answers:
- Where are we over-discounting?
- Which departments drive margin leakage?
- Which customer segments actually need discounts?
